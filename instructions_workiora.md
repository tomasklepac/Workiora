# Project: Workiora – Web Presentation

## Kontext projektu
Tento projekt slouží k vytvoření moderní, reprezentativní webové prezentace pro společnost **Workiora**, která působí jako personální agentura zaměřená výhradně na zajištění personálu pro eventové akce.

Web je první reálný klientský projekt v rámci rozjezdu freelance web developmentu Tomáše Klepače. Cílem je dodat profesionální, vizuálně silný, ale technicky jednoduchý web (statický web), který bude sloužit jako oficiální vizitka firmy.

---

## Klient – základní informace

**Název:** Workiora  
**Web / doména:** https://www.workiora.com  
**Instagram:** https://www.instagram.com/workiora  

### Charakter firmy
- Personální eventová agentura
- Funguje cca 6 let
- Dodává **výhradně personál**, nikoli produkci akcí
- Nespadá pod:
  - ozvučení
  - techniku
  - produkci eventů

### Typy personálu
- Hostesky a promotéři
- Stagehands (technická výpomoc)
- Bezpečnostní služby / security
- Další eventový personál dle požadavků klienta

---

## Cíl webu

- Reprezentativní prezentace firmy
- Jasné vymezení služeb a kompetencí
- Podpora důvěry (zkušenosti, reference)
- Silný vizuální dojem (fotky z akcí)
- Propojení se sociálními sítěmi (Instagram)

Web **není**:
- e-shop
- rezervační systém
- aplikace

Jedná se o **statický prezentační web**.

## Technické předpoklady

- Web bude realizován jako:
  - HTML + CSS
  - případně JS pro drobné interakce
- Je možné použít libovolný CSS framework / utility framework dle uvážení agenta (např. Tailwind, Bootstrap, vlastní CSS)
- Žádné CMS (WordPress apod.)
- Žádný backend

### Hosting & nasazení

- Finální hosting bude pravděpodobně na **cizím hostingu klienta**
- Vývoj však musí probíhat:
  - lokálně
  - nebo na **náhledové subdoméně developera** (např. preview.tomasklepac.cz/workiora)

➡️ Finální nahrání na klientský hosting proběhne **až po schválení a zaplacení**.

## Vizuální směr

- Dark / modern theme
- Velké hero fotografie z eventů
- Jednoduchá, přehledná typografie
- Jemné animace (fade-in, scroll), žádné heavy efekty

Barevnost (orientačně):
- tmavá šedá / černá
- bílý text
- jeden decentní akcent (např. zlatá / studená béžová / modrá)

## Navržená struktura webu

### 1. Hero sekce
- Full-width fotografie z eventu
- Jasný headline typu:
  - „Personální zajištění eventových akcí"
- Podtitulek (hostesky, stagehands, security)
- CTA tlačítka:
  - Naše služby
  - Kontakt

### 2. O nás
- Stručný popis firmy
- Zvýraznění:
  - 6+ let zkušeností
  - specializace na eventový personál
  - spolupráce s agenturami a produkcemi
- Výrazné sdělení:
  - ❌ nezajišťujeme techniku, ozvučení ani produkci
  - ✔️ dodáváme výhradně personál

### 3. Služby
Karty / sekce:
- Hostesky & promotéři
- Stagehands
- Bezpečnostní služby
- Eventový personál na míru

Každá služba:
- fotografie
- krátký popis (1–2 věty)

### 4. Reference / Galerie
- Grid nebo masonry galerie
- Autentické fotky z akcí
- Bez dlouhých popisků – čistě vizuální důkaz

### 5. Sociální sítě
- Propojení na Instagram
- CTA: „Sledujte nás na Instagramu"
- případně embed IG feed (není nutné)

### 6. Kontakt
- E-mail
- Telefon
- Instagram
- Krátká výzva k poptávce
